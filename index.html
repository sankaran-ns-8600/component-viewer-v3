The complete attachment viewer
for modern jQuery apps

A full-screen overlay for previewing images, video, audio, PDFs, code, markdown, and HTML — with zoom/pan, keyboard nav, carousel, slideshow, OCR text extraction, and WCAG accessibility on by default.

Features

Everything you need for previewing files

Modular renderers, opt-in features, and a clean API. Pick what you use, ship a smaller bundle, and override anything.

Images with zoom & pan

Mouse wheel zoom, pinch on touch, drag to pan. Hi-res swap supported on open. Landscape and portrait both fit beautifully.

Video & audio (jPlayer)

Rich player via jPlayer when loaded; native HTML5 fallback otherwise. HD switch, autoplay-on-open, keyboard seek.

PDF rendering

PDF.js integration with page nav, zoom, rotate, thumbnails, two-page view, and auto-fit. Iframe fallback when PDF.js is absent.

Code & Markdown

Inline code with line numbers and syntax highlighting. Markdown via marked.js, with safe textContent rendering by default.

Keyboard & A11y

Focus trap, ARIA labels, and customizable shortcut keys per toolbar item. Press ? to see all shortcuts.

Modular & extensible

Use the full bundle for v2 drop-in compat, or include only the renderers and features you need. Override any type via onRender.

Showcase

Built for every kind of media

From a single image to a full-blown PDF document, the viewer adapts to the content.

Full-screen overlay with zoom & pan

Click any attachment to open it in a polished full-screen overlay. Smooth zoom, drag-to-pan, and hi-res swap on open.

PDF viewer

Page navigation, thumbnails, rotate, zoom, two-page spread, and auto-fit — all powered by PDF.js.

Video & audio playback

jPlayer for rich playback when loaded; native HTML5 fallback otherwise. HD toggle, posters, and keyboard seek.

Thumbnail carousel

Drop in a strip of thumbnails for navigating multi-attachment posts. Pairs with auto-advance slideshow mode.

Light & dark themes

Drops into your existing UI. Toggle themes from the toolbar or set a default; press T to switch.

Get started

Up and running in under a minute

Pick the bundle that fits — full-feature drop-in or modular for the smallest possible footprint.

Full bundle (drop-in)

Same API as v2. Works with markup that uses .cv-item.

<link rel="stylesheet"
      href="component-viewer.min.css">
<script src="jquery.min.js"></script>
<script src="component-viewer.min.js"></script>

<script>
  $('#gallery').componentViewer({
    toolbar: { download: true, zoom: true }
  });
</script>

Modular (pick what you need)

Tree-shake at the script-tag level. Core + image is ~2.5 KB gzipped.

<link rel="stylesheet"
      href="component-viewer.min.css">
<script src="jquery.min.js"></script>
<script src="src/utils.js"></script>
<script src="src/core.js"></script>
<script src="src/renderer-image.js"></script>
<!-- Add only the renderers you use -->
<script src="src/renderer-pdf.js"></script>
Highlights

Built for modern jQuery apps

Cleaner architecture, smaller footprint, safer defaults — modular by design.

Modular renderers

Include only the renderers you use, or use the full bundle for drop-in v2 compatibility.

WCAG on by default

Focus trap, ARIA attributes, and keyboard labels are enabled out of the box.

Safer markdown

Markdown rendered through textContent by default; plug in DOMPurify for raw HTML when needed.

Smaller footprint

Core + image renderer is ~2.5 KB gzipped. Add only what you use, ship less to your users.

Cleaner internals

Renderers receive inst and overlay as parameters — explicit, testable, no globals.

Pure CSS

Removed Less-only syntax. Reduced !important usage. Drops in to plain stylesheets.
